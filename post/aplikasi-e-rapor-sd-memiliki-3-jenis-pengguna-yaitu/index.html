Aplikasi E-Rapor Sd Memiliki 3 Jenis Pengguna Yaitu. Pada artikel ini admin akan sedikit memberikan panduan singkat mengenai pada aplikasi erapor sd ini telah disediakan fitur untuk mengambil semua data dari aplikasi dapodik, untuk caranya silahkan simak dibawah ini. 1.0 di sd dilakukan secara bertahap sesuai dengan kesiapan sumberdaya 5.

Kemdikbud memberikan hak kepada pengguna perangkat lunak dalam hal ini sekolah dasar, dinas pendidikan kabupaten atau kota, dinas pendidikan provinsi. 1.0 di sd dilakukan secara bertahap sesuai dengan kesiapan sumberdaya 5. Input kehadiran siswa, nilai ekstra kurikuler, proses deskripsi sikap, status penilaian, cetak rapor, menambah foto siswa, dan mengganti logo sekolah.
